# Star Wars Planets
Star Wars planets rendered randomly in a card. Made with [Vue.js](https://vuejs.org/) and [The Star Wars API](https://swapi.co/).

### Setup
```sh
$ npm install
```
### Compiles and hot-reloads for development
```sh
$ npm run serve
```
### Compiles and minifies for production
```sh
$ npm run build
```
